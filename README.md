## Getting Started

This is a [Next.js](https://nextjs.org) project bootstrapped with `create-next-app`.

### 1. Run the development server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
